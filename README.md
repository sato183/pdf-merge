# pdf-merge

複数のPDFファイルを1つのPDFファイルに結合するシンプルなツールです。

## 機能

- 指定したディレクトリ（`input_pdfs`）内の全てのPDFファイルを読み込みます。
- ファイル名順にソートして結合します。
- 結合されたPDFを `merged_document.pdf` として出力します。

## 使い方

1. このリポジトリをクローンまたはダウンロードします。
2. 依存関係をインストールします。

```bash
npm install
```

3. 結合したいPDFファイルを `input_pdfs` ディレクトリに入れます。（ディレクトリがない場合は作成してください）

4. スクリプトを実行します。

```bash
npm start
```
